## React + TypeScript + RTL
- Todolist를 만들며 타입스크립트와 유닛테스트를 연습한다.
- 테스트코드가 없는 기능은 구현하지 않는다.
- UI에 집중하는 React의 철학에 RTL이 더욱 부합한다고 느껴 Jest + Enzyme 조합에서 RTL로 선회함.

## 적용 기술
- React.js
- Hooks
- Styled components
- TypeScript
- React Testing Library(based on Jest)
